# @pixelwolf/tsconfig

###Installation

`npm i @pixelwolf/tsconfig --save-dev`

### Usage

On `tsconfig.json` file add:

**React or vanilla Typescript:**

```json
{
  "extends": "./node_modules/@pixelwolf/tsconfig/index.json"
}
```

### Script in package.json

```json
"scripts": {
  "build": "tsc"
},
```
